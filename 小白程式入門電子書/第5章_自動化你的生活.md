## 第一個自動化：批次改名

阿威遇到了一個問題。客戶傳來200張產品照片，檔名亂七八糟：
- IMG_001.jpg
- 照片1.png
- 未命名-2.jpeg
- Copy of DSC1234.JPG

他需要統一改成：
- 產品_001.jpg
- 產品_002.jpg
- 產品_003.jpg
- ...

手動改？要改到什麼時候？

我教他寫了第一個自動化程式：

```python
import os

資料夾路徑 = "產品照片"
檔案列表 = os.listdir(資料夾路徑)
計數器 = 1

for 舊檔名 in 檔案列表:
    if 舊檔名.endswith(('.jpg', '.jpeg', '.png')):
        新檔名 = f"產品_{計數器:03d}.jpg"
        os.rename(舊檔名, 新檔名)
        計數器 += 1
        print(f"已改名：{舊檔名} → {新檔名}")
```

五秒鐘，200個檔案改好了。

阿威盯著螢幕，久久說不出話。然後他說了一句我永遠不會忘記的話：

「這三行程式碼，抵得上我兩個小時的工作。」

不，是抵得上他未來所有需要改檔名的時間。因為程式寫一次，可以用一輩子。

## Excel自動化：解放小美

回到小美的週一地獄。

我們一起分析了她的工作流程，發現全部都可以自動化：

```python
import pandas as pd
import glob

# 找出所有分店的報表
所有報表 = glob.glob("分店報表/*.xlsx")

# 創建一個空的總表
總表 = pd.DataFrame()

# 讀取每個報表，合併到總表
for 報表路徑 in 所有報表:
    分店名稱 = 報表路徑.split("/")[-1].replace(".xlsx", "")

    # 讀取Excel
    資料 = pd.read_excel(報表路徑)

    # 加上分店名稱
    資料["分店"] = 分店名稱

    # 合併到總表
    總表 = pd.concat([總表, 資料])

# 計算各種統計
總銷售額 = 總表["銷售額"].sum()
平均客單價 = 總表["銷售額"].mean()
最佳產品 = 總表.groupby("產品")["數量"].sum().idxmax()

# 輸出結果
總表.to_excel("週報總表.xlsx", index=False)
print(f"本週總銷售額：{總銷售額:,.0f}")
print(f"平均客單價：{平均客單價:.2f}")
print(f"最暢銷產品：{最佳產品}")
```

第一次執行成功時，小美的反應是：「等等...就這樣？三小時的工作，10秒就完成了？」

是的，就這樣。

而且不只是快，還更準確。沒有複製貼上的錯誤，沒有公式拉錯，沒有漏掉的分店。

## 網路爬蟲：資訊自動收集

小李的工作是市場研究，需要每天查看競爭對手的價格。

以前的流程：
1. 打開10個競品網站
2. 搜尋產品
3. 記錄價格
4. 輸入Excel
5. 每天重複

這不只無聊，還容易出錯。看錯一個數字，整個分析就歪了。

自動化解決方案：

```python
import requests
from bs4 import BeautifulSoup
import schedule

def 檢查競品價格():
    競品網址 = [
        "https://competitor1.com/product",
        "https://competitor2.com/product",
        # ...
    ]

    今日價格 = {}

    for 網址 in 競品網址:
        回應 = requests.get(網址)
        網頁 = BeautifulSoup(回應.text, 'html.parser')

        # 找出價格（每個網站結構不同）
        價格元素 = 網頁.find('span', class_='price')
        價格 = float(價格元素.text.replace('$', ''))

        商家名稱 = 網址.split('/')[2]
        今日價格[商家名稱] = 價格

    # 儲存到資料庫或Excel
    儲存價格(今日價格)

    # 如果有異常變動，發出警報
    for 商家, 價格 in 今日價格.items():
        if 價格變動超過10%:
            發送警報(f"{商家}價格異常變動！")

# 每天早上9點自動執行
schedule.every().day.at("09:00").do(檢查競品價格)
```

現在小李每天早上打開電腦，價格報告已經在信箱裡了。如果有重要變動，手機會立刻收到通知。

「我從資料收集員變成了資料分析師，」他說，「這才是我真正的價值所在。」

## 檔案整理：數位斷捨離

阿威的電腦桌面是一場災難。幾百個檔案散落各處，要找東西全靠搜尋。

我們寫了一個自動整理程式：

```python
import os
import shutil
from datetime import datetime

def 整理桌面():
    桌面路徑 = os.path.expanduser("~/Desktop")

    # 建立分類資料夾
    分類 = {
        "圖片": ['.jpg', '.png', '.gif', '.psd'],
        "文件": ['.doc', '.docx', '.pdf', '.txt'],
        "試算表": ['.xls', '.xlsx', '.csv'],
        "影片": ['.mp4', '.mov', '.avi'],
        "其他": []
    }

    # 建立資料夾
    for 資料夾名稱 in 分類.keys():
        資料夾路徑 = os.path.join(桌面路徑, 資料夾名稱)
        if not os.path.exists(資料夾路徑):
            os.makedirs(資料夾路徑)

    # 整理檔案
    for 檔案名稱 in os.listdir(桌面路徑):
        if os.path.isfile(os.path.join(桌面路徑, 檔案名稱)):
            檔案分類 = "其他"

            for 類別, 副檔名列表 in 分類.items():
                if any(檔案名稱.endswith(ext) for ext in 副檔名列表):
                    檔案分類 = 類別
                    break

            # 移動檔案
            來源 = os.path.join(桌面路徑, 檔案名稱)
            目標 = os.path.join(桌面路徑, 檔案分類, 檔案名稱)
            shutil.move(來源, 目標)
            print(f"已移動：{檔案名稱} → {檔案分類}/")
```

執行後，混亂的桌面瞬間井然有序。圖片在圖片資料夾，文件在文件資料夾。

「這就像請了一個數位管家，」阿威說。

## 定時任務：讓電腦記得你會忘記的事

人會忘記，電腦不會。

小美常常忘記備份重要檔案，直到有一天硬碟壞了，她才後悔莫及。

現在，她有了自動備份：

```python
import shutil
import schedule
from datetime import datetime

def 自動備份():
    重要資料夾 = [
        "工作文件",
        "客戶資料",
        "財務報表"
    ]

    今天日期 = datetime.now().strftime("%Y%m%d")
    備份資料夾 = f"備份_{今天日期}"

    for 資料夾 in 重要資料夾:
        來源 = f"C:/Users/小美/{資料夾}"
        目標 = f"D:/備份/{備份資料夾}/{資料夾}"

        shutil.copytree(來源, 目標)
        print(f"已備份：{資料夾}")

    # 刪除30天前的備份（節省空間）
    清理老舊備份()

# 每天下班前自動備份
schedule.every().day.at("17:30").do(自動備份)

# 每週五額外備份到雲端
schedule.every().friday.at("18:00").do(備份到雲端)
```

現在她再也不用擔心資料遺失了。

## 自動化的哲學

自動化不只是技術，更是一種思維方式。

當你開始用自動化的眼光看世界，你會發現處處都有機會：
- 每天檢查的網站 → RSS訂閱或爬蟲
- 定期發送的報告 → 排程任務
- 重複的滑鼠點擊 → 腳本程式
- 例行的資料處理 → 批次處理

但自動化的真正價值，不在於省下的時間，而在於釋放的心智。

當你不需要記住「每週一要整理報表」，你的大腦就有空間思考「如何提升銷售」。

當你不需要手動備份檔案，你就不會焦慮資料安全。

當重複的工作都自動化了，你終於可以專注在真正重要的事：創造、思考、連結。
