## 教學設計特色

1. **金字塔式知識架構**：從工具 -> 技能 -> 思維 -> 影響力逐層遞進
2. **Excel 錨點教學法**：每個 Python 概念都對照 Excel 操作建立心智模型
3. **商業場景驅動**：不教語法，教解決商業問題
4. **實戰貫穿**：同一份超市銷售資料集貫穿 M2~M7，體驗完整分析生命週期
5. **模組化成就系統**：每個模組結尾有 Learning Checklist + 立即行動任務
6. **AI 原生整合**：將 Vibe Coding 與 AI 工具作為獨立模組納入數據分析工作流
