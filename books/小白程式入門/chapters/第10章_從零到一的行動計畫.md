## 第一天：選擇你的武器

不要糾結太久。記住第二章說的——語言是工具，不是目的。

```
如果你是：
小美（行銷/業務）→ Python

阿威（設計/創意）→ JavaScript
理由：網頁互動必備，視覺回饋即時

老陳（金融/分析）→ Python
理由：數據分析首選，函式庫豐富

小李（想轉職）→ JavaScript 或 Python

還是不確定？→ Python
理由：最適合初學者的語言
```

今天的任務：
1. 安裝Python（去python.org）或Node.js（去nodejs.org）
2. 安裝VS Code（去code.visualstudio.com）
3. 寫下你的第一行程式碼：`print("我開始了")`

就這樣。第一天不要貪心。

## 第一週：建立習慣

習慣比天賦更重要。

我見過太多人第一天學8小時，然後消失三個月。別這樣。

**每日30分鐘計畫：**

```
週一：變數與資料型態
範例：姓名 = "你的名字"
      年齡 = 你的年齡
      print(f"我是{姓名}，今年{年齡}歲")

週二：條件判斷
範例：如果年齡 >= 18:
          print("成年人")
      否則:
          print("未成年")

週三：迴圈
範例：for 次數 in range(10):
          print(f"第{次數+1}次練習")

週四：函數
範例：def 問候(名字):
          return f"你好，{名字}！"

週五：綜合練習
任務：寫一個猜數字遊戲

週末：回顧與整理筆記
```

關鍵：每天都要碰程式碼，哪怕只有10分鐘。

## 第一個月：打地基

四週，四個里程碑。

**第1週：基礎語法**
- 目標：能寫出簡單的互動程式
- 專案：計算機、單位轉換器
- 心態：允許自己犯錯

**第2週：資料處理**
- 目標：能讀寫檔案，處理資料
- 專案：通訊錄管理、CSV處理
- 心態：享受自動化的快感

**第3週：函數與模組**
- 目標：能組織程式碼，重複使用
- 專案：待辦事項App
- 心態：追求簡潔優雅

**第4週：第一個完整專案**
- 目標：完成一個自己會用的工具
- 專案：個人記帳本（像第六章）
- 心態：完成比完美重要

月底檢視：
- 寫了多少行程式碼？
- 解決了什麼實際問題？
- 下個月想學什麼？

## 避開陷阱

我看過太多人在這些地方跌倒：

**陷阱1：教學地獄**
```
症狀：看了100個教學，沒寫過一個專案
解方：看一個教學，做三個練習
```

**陷阱2：完美主義**
```
症狀：程式碼改了又改，專案永遠沒完成
解方：設定期限，時間到就發布
```

**陷阱3：比較心態**
```
症狀：看到別人的作品就沮喪
解方：只跟昨天的自己比
```

**陷阱4：獨自奮戰**
```
症狀：遇到問題就卡住，沒人討論
解方：加入社群，找學習夥伴
```

**陷阱5：貪多嚼不爛**
```
症狀：同時學5種語言，每種都半調子
解方：專注一個，學到能用
```

## 建立你的學習系統

成功的學習者都有系統。

### 每日儀式

```python
def 每日學習():
    # 早上（5分鐘）
    複習昨天的程式碼()

    # 通勤（15分鐘）
    閱讀技術文章()

    # 午休（20分鐘）
    寫一個小功能()

    # 晚上（30分鐘）
    專案開發()

    # 睡前（10分鐘）
    寫學習日記()
```

### 週末專案日

每個週末，給自己2-4小時的完整時間。

不是學習，而是創造。

用這週學的東西，做一個小作品。

### 月度回顧

每個月最後一天，問自己：
- 這個月學了什麼？
- 寫了哪些程式？
- 解決了什麼問題？
- 下個月要挑戰什麼？
