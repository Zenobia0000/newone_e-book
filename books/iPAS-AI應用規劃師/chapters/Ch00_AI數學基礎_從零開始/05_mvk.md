## 1 必背公式速查（12 條）

### Part A：機率與統計

```
① 條件機率
   P(A|B) = P(A∩B) / P(B)
   ⚠️ P(A|B) ≠ P(B|A)

② 貝氏定理
   P(A|B) = P(B|A) × P(A) / P(B)
   口訣：後驗 = 似然 × 先驗 / 證據

③ 期望值
   E(X) = Σ xᵢ × P(xᵢ)    （離散）
   = 加權平均

④ 變異數
   Var(X) = E[(X - μ)²] = E(X²) - [E(X)]²
   = 離散程度

⑤ Z-score（→ Ch08 會用到）
   Z = (X - μ) / σ
   = 離均值幾個標準差
```

### Part B：線性代數

```
⑥ 內積
   a · b = Σ aᵢbᵢ = |a||b|cosθ
   cosθ 越接近 1 = 越相似

⑦ L2 範數
   ||x||₂ = √(Σ xᵢ²)
   = 向量長度（畢氏定理推廣）

⑧ 矩陣乘法維度規則
   (m, n) × (n, p) = (m, p)
   ⚠️ 中間的 n 必須一樣！
```

### Part C：最佳化

```
⑨ MSE（迴歸損失）
   L = (1/n) Σ(y - ŷ)²

⑩ Cross-Entropy（分類損失）
   L = -Σ yᵢ log(ŷᵢ)

⑪ 梯度下降更新規則
   w = w - lr × ∇L
   lr 太大 → 震盪 / lr 太小 → 太慢

⑫ 正則化
   L1 penalty = λΣ|wᵢ|  → 稀疏（鑽石形）
   L2 penalty = λΣwᵢ²   → 小權重（圓形）
```

## 5 Part A/B/C 各自一句話收束

- **Part A 機率統計**：用機率量化不確定性 → 貝氏定理翻轉直覺、Type I/II 守住判斷。
- **Part B 線性代數**：用矩陣搬運資料 → 內積 = 預測、矩陣乘法 = 前向傳播。
- **Part C 最佳化**：用梯度下降找答案 → MSE/CE 定義目標、lr 控制步伐、L1/L2 防止過擬合。

## 6 考前最後 5 分鐘自測

1. P(A|B) 和 P(B|A) 相等嗎？ → **不相等**，除非 P(A)=P(B)。
2. (3,4) × (4,2) = ? → **(3,2)**。
3. MSE 用在什麼任務？CE 呢？ → **MSE=迴歸，CE=分類**。
4. lr 太大的症狀？ → **loss 震盪不收斂**。
5. L1 正則化產生什麼效果？ → **稀疏（很多權重=0）**。
6. Type I 錯誤的比喻？ → **喊狼來了（假陽性）**。
7. 梯度下降的更新規則？ → **w = w - lr × ∇L**。
8. PCA 用什麼數學工具降維？ → **協方差矩陣的最大特徵值方向**。
