## 五、模擬試題（10 題）

### 模擬題 1（類型 A — 條件機率方向）

下列關於條件機率的描述，何者正確？

(A) P(A|B) 一定等於 P(B|A)
(B) P(有病|陽性) 等於 P(陽性|有病)
(C) P(A|B) = P(A∩B) / P(B)，前提是 P(B) > 0
(D) 條件機率的大小不受事前機率 P(A) 的影響

**答案：(C)**

**詳解**：條件機率的定義是 P(A|B) = P(A∩B) / P(B)，前提是 P(B) > 0（分母不能為零）。(A)(B) 都犯了方向錯誤——P(A|B) 不等於 P(B|A)，除非 P(A)=P(B)。例如 P(有病|陽性) 可能只有 17%，但 P(陽性|有病) 可能是 99%。(D) 貝氏定理明確告訴我們事前機率 P(A) 會影響後驗機率。

### 模擬題 2（類型 C — 貝氏定理計算）

某疾病的盛行率為 2%，檢測的靈敏度(Sensitivity)為 95%，特異度(Specificity)為 90%。一個人檢測結果為陽性，他真正有病的機率最接近下列何者？

(A) 95%
(B) 66%
(C) 16%
(D) 2%

**答案：(C)**

**詳解**：使用貝氏定理。P(Disease) = 0.02，P(+|D) = 0.95，P(+|~D) = 1-0.90 = 0.10。
P(+) = P(+|D)P(D) + P(+|~D)P(~D) = 0.95×0.02 + 0.10×0.98 = 0.019 + 0.098 = 0.117。
P(D|+) = 0.019 / 0.117 ≈ 0.162 ≈ 16%。
即使檢測靈敏度高達 95%，因為盛行率只有 2%，陽性預測值只有約 16%。這是貝氏定理的經典結果。(A) 是混淆了靈敏度和陽性預測值。

---

### 模擬題 3（類型 B — Type I/II 錯誤）

某製造工廠的品管系統在檢測產品瑕疵時，偶爾會將合格品誤判為瑕疵品而丟棄。這屬於下列何種錯誤？

(A) Type I 錯誤（偽陽性）
(B) Type II 錯誤（偽陰性）
(C) 系統性誤差
(D) 抽樣誤差

**答案：(A)**

**詳解**：H0（虛無假設）= 產品合格。將合格品誤判為瑕疵品 = 拒絕了正確的 H0 = Type I 錯誤（偽陽性）。Type II 錯誤是反過來：瑕疵品沒被抓到，當作合格品出貨。口訣：Type I = 喊狼來了（其實沒有問題）；Type II = 放走壞人（有問題但沒發現）。

### 模擬題 4（類型 C — 矩陣乘法維度）

矩陣 A 的 shape 為 (3, 4)，矩陣 B 的 shape 為 (4, 2)。計算 A × B 的結果矩陣 shape 為何？

(A) (3, 2)
(B) (4, 4)
(C) (3, 4)
(D) (2, 3)

**答案：(A)**

**詳解**：矩陣乘法維度規則：(m, n) × (n, p) = (m, p)。A 是 (3, 4)，B 是 (4, 2)。中間的 4 匹配，所以可以相乘。結果取兩端 = (3, 2)。考試的矩陣乘法維度判斷就是這個規則，沒有例外。

### 模擬題 5（類型 C — 矩陣乘法手算）

A = [[1, 0, 2], [3, 1, 0]]，B = [[2, 1], [0, 3], [1, 0]]。計算 A × B 的結果矩陣中，第一行第二列的值為何？

(A) 1
(B) 4
(C) 5
(D) 7

**答案：(A)**

**詳解**：結果矩陣 C 的 (0,1) 位置 = A 的第一行 [1, 0, 2] 與 B 的第二列 [1, 3, 0] 做內積 = 1×1 + 0×3 + 2×0 = 1。驗算其他位置：C(0,0) = 1×2 + 0×0 + 2×1 = 4；C(1,0) = 3×2 + 1×0 + 0×1 = 6；C(1,1) = 3×1 + 1×3 + 0×0 = 6。

### 模擬題 6（類型 A — 損失函數選擇）

下列何者是分類任務最常使用的損失函數？

(A) Mean Squared Error (MSE)
(B) Cross-Entropy Loss
(C) Mean Absolute Error (MAE)
(D) Huber Loss

**答案：(B)**

**詳解**：Cross-Entropy Loss 是分類任務的標準損失函數，它衡量預測機率分佈與真實標籤分佈之間的差距。MSE、MAE、Huber Loss 都是迴歸任務的損失函數。口訣：迴歸 = MSE，分類 = Cross-Entropy。考試常用這個對比來出題。

### 模擬題 7（類型 B — Learning Rate 判斷）

在訓練神經網路時，觀察到 loss 值在每個 epoch 之間劇烈上下震盪且不收斂。最可能的原因是下列何者？

(A) Learning rate 太小
(B) Learning rate 太大
(C) 訓練資料不足
(D) 模型太簡單

**答案：(B)**

**詳解**：Loss 劇烈震盪 = learning rate 太大。步伐太大導致在最低點附近跳來跳去，無法收斂。lr 太小的症狀是 loss 下降非常緩慢但不會震盪。訓練資料不足通常表現為 overfitting（訓練 loss 低但驗證 loss 高），不是震盪。模型太簡單會導致 underfitting（loss 居高不下但穩定）。

### 模擬題 8（類型 B — L1 vs L2 正則化）

關於 L1 正則化（Lasso）與 L2 正則化（Ridge），下列描述何者正確？

(A) L1 正則化傾向使權重趨近於零但不為零
(B) L2 正則化會產生稀疏的權重向量（很多權重恰好為零）
(C) L1 正則化的約束區域為圓形，L2 為菱形
(D) L1 正則化傾向產生稀疏解（很多權重恰好為零），L2 傾向使所有權重均勻縮小

**答案：(D)**

**詳解**：L1 正則化的約束區域是菱形（鑽石形），菱形的角在座標軸上，最佳解容易落在角上 → 某些權重恰好 = 0 → 稀疏。L2 正則化的約束區域是圓形，解落在圓上任意位置 → 權重會縮小但不會變 0。(A) 描述的是 L2 的效果。(B) 描述的是 L1 的效果。(C) 把兩者的幾何形狀搞反了。

---

### 模擬題 9（類型 B — 凸性判斷）

下列關於凸函數與非凸函數的描述，何者正確？

(A) 線性回歸的 MSE 損失函數是非凸函數
(B) 神經網路的損失函數是凸函數，一定能找到全域最佳解
(C) 凸函數只有一個最小值（全域最小），梯度下降一定能找到
(D) 非凸函數不能使用梯度下降

**答案：(C)**

**詳解**：凸函數的定義特性就是只有一個最小值（全域最小），不存在其他局部最小。線性回歸的 MSE 就是凸函數，梯度下降一定能找到最佳解。(A) 線性回歸的 MSE 是凸函數，不是非凸。(B) 神經網路的損失函數是非凸的，有多個局部最小值。(D) 非凸函數仍然可以用梯度下降，只是不保證找到全域最佳——可能卡在局部最小。

### 模擬題 10（類型 D — 前向傳播維度推算）

一個神經網路層的輸入為 5 個特徵，輸出為 3 個神經元。若一批次有 10 筆資料，下列關於權重矩陣 W 和輸出矩陣的維度描述，何者正確？

(A) W 的 shape 為 (3, 5)，輸出 shape 為 (10, 3)
(B) W 的 shape 為 (5, 3)，輸出 shape 為 (10, 3)
(C) W 的 shape 為 (5, 3)，輸出 shape 為 (3, 10)
(D) W 的 shape 為 (10, 3)，輸出 shape 為 (10, 5)

**答案：(B)**

**詳解**：前向傳播 z = X × W + b。輸入 X 的 shape 為 (batch, d_in) = (10, 5)。權重 W 的 shape 為 (d_in, d_out) = (5, 3)。根據矩陣乘法 (10, 5) × (5, 3) = (10, 3)，所以輸出 shape 為 (10, 3)。注意 W 的第一維必須等於輸入的特徵數(5)，第二維是輸出的神經元數(3)。
