## 3. 關鍵教學點

### Part A：機率與統計基礎（60 min）

#### 隨機變數

- **離散隨機變數**：值可以列舉（骰子 1-6、硬幣正反）。
  - 機率表示：P(X = x)，所有可能值的機率加總 = 1。
  - AI 應用：分類模型的輸出就是離散的。

- **連續隨機變數**：值不可列舉（身高、體重、溫度）。
  - 機率表示：用機率密度函數 f(x)，P(a < X < b) = 積分 f(x)dx。
  - AI 應用：迴歸模型的輸出就是連續的。

- **教學提醒**：不需要教積分計算，只需要知道「連續變數不能問 P(X=某個值)，要問 P(X 在某範圍內)」。

#### 條件機率

- **公式**：P(A|B) = P(A 且 B) / P(B)
- **關鍵陷阱**：P(A|B) 不等於 P(B|A)。
  - 例：P(有病|陽性) 不等於 P(陽性|有病)。
  - 前者是你拿到報告後最想知道的，後者是醫學文獻的靈敏度。
- **討論提示**：「你的 COVID 快篩陽性了，你真的有病的機率是多少？」→ 引出貝氏定理。

#### 貝氏定理（核心案例：醫療篩檢）

- **公式**：P(A|B) = P(B|A) x P(A) / P(B)
- **案例完整推導**：
  - 疾病盛行率 P(Disease) = 1%
  - 靈敏度 Sensitivity = P(陽性|有病) = 99%
  - 特異度 Specificity = P(陰性|沒病) = 95%
  - 求：P(有病|陽性) = ?
  - P(陽性) = P(陽性|有病)xP(有病) + P(陽性|沒病)xP(沒病) = 0.99x0.01 + 0.05x0.99 = 0.0594
  - P(有病|陽性) = 0.99x0.01 / 0.0594 = 0.167 = 約 17%
- **教學重點**：盛行率低的時候，即使檢測很準，陽性結果的真實性也不高。這就是為什麼要做二次篩檢。
- **AI 連結**：Naive Bayes 分類器的核心就是貝氏定理（→ Ch09）。

#### 期望值與變異數

- **期望值**：E(X) = Sigma x_i x P(x_i)（離散）= 加權平均。
  - AI 連結：損失函數 = 預測誤差的期望值。
  - 例：骰子 E(X) = (1+2+3+4+5+6)/6 = 3.5。

- **變異數**：Var(X) = E[(X - mu)^2] = E(X^2) - [E(X)]^2。
  - 意義：資料離散程度。Var 大 = 不穩定。
  - AI 連結：模型的 variance 高 = overfit。
  - 幾何直覺：期望值 = 分佈的中心位置，變異數 = 分佈的寬度。

#### 中央極限定理（CLT）

- **一句話**：不管母體什麼分佈，樣本均值的分佈會趨近常態分佈（n >= 30 就差不多了）。
- **為什麼重要**：假設檢定能用常態分佈的前提就是 CLT。
- **教學提醒**：只需一張圖——展示不同母體分佈的抽樣均值都趨近常態。不需推導。

#### Type I/II 錯誤

- **Type I 錯誤 (alpha)**：H0 是對的，但你拒絕了它。
  - 比喻：喊「狼來了」但沒有狼。
  - alpha = 容忍 Type I 錯誤的上限（通常 0.05）。

- **Type II 錯誤 (beta)**：H0 是錯的，但你沒有拒絕它。
  - 比喻：狼真的來了但你沒喊。
  - Power = 1 - beta = 抓到真正差異的能力。

- **p-value**：在 H0 為真的假設下，觀察到這麼極端結果的機率。
  - p < alpha → 拒絕 H0。
  - p >= alpha → 不拒絕 H0（不是「接受 H0」）。

- **考試重點**：給情境判斷 Type I/II、給 p-value 和 alpha 做決策。

### Part B：線性代數（50 min）

#### 向量

- **定義**：一組有序數字的清單。x = [x1, x2, ..., xd]。
- **AI 意義**：一個樣本的 d 個特徵 = d 維向量。
- **視覺**：2D 中的箭頭。[3, 4] = 從原點到 (3,4) 的箭頭。
- **教學提醒**：用試算表比喻——一行資料 = 一個向量。

#### 內積（Dot Product）

- **公式**：a . b = Sigma a_i x b_i = |a| |b| cos(theta)。
- **幾何意義**：投影長度 x 另一向量長度。cos(theta) = 相似度。
- **手算練習**：[1, 2, 3] . [4, 5, 6] = 1x4 + 2x5 + 3x6 = 4 + 10 + 18 = 32。
- **AI 連結**：線性回歸的預測 y_hat = w . x + b 就是內積（→ Ch09）。

#### L2 範數

- **公式**：||x||_2 = sqrt(x1^2 + x2^2 + ... + xd^2)。
- **直覺**：就是畢氏定理在高維的推廣 = 向量長度。
- **AI 連結**：L2 正則化 = 限制權重向量的長度（→ Part C）。

#### 矩陣

- **定義**：n 個向量疊在一起 = n 行 d 列的矩陣，shape = (n, d)。
- **視覺**：試算表就是矩陣。100 筆資料 x 5 個特徵 = shape(100, 5)。
- **教學提醒**：強調 shape 的表示法，因為 Python 程式碼裡到處都是 shape。

#### 矩陣乘法（考試直接考）

- **維度規則**：(m, n) x (n, p) = (m, p)。中間的 n 必須一樣，結果取兩端。
- **手算步驟**：結果的 (i, j) = 第一個矩陣的第 i 行 dot 第二個矩陣的第 j 列。
- **完整手算範例**：
  ```
  A = [[1, 2, 3],    B = [[7, 8],
       [4, 5, 6]]         [9, 10],
                           [11, 12]]

  A(2,3) x B(3,2) = C(2,2)

  C[0,0] = 1x7 + 2x9 + 3x11 = 7 + 18 + 33 = 58
  C[0,1] = 1x8 + 2x10 + 3x12 = 8 + 20 + 36 = 64
  C[1,0] = 4x7 + 5x9 + 6x11 = 28 + 45 + 66 = 139
  C[1,1] = 4x8 + 5x10 + 6x12 = 32 + 50 + 72 = 154
  ```
- **考試天花板**：樣題只考 2x3 矩陣乘法，不教 4x4。

#### 前向傳播

- **公式**：z = Wx + b → a = activation(z)。
- **維度**：輸入 (batch, d_in) x 權重 (d_in, d_out) + 偏差 (d_out) = 輸出 (batch, d_out)。
- **教學提醒**：這裡只建立概念，完整手算在 Ch09 S17。

#### 特徵值/特徵向量

- **定義**：Av = lambda x v。矩陣作用在某些特殊向量上只改變長度不改變方向。
- **視覺**：矩陣 = 拉伸/旋轉。特徵向量 = 只被拉伸的方向。特徵值 = 拉伸倍率。
- **AI 連結**：PCA = 找協方差矩陣的最大特徵值方向 = 資料變異最大的方向（→ Ch08）。
- **考試不考計算**：只考概念——「PCA 為什麼能降維？因為沿最大特徵值方向投影保留最多資訊。」

### Part C：最佳化與梯度下降（50 min）

#### 損失函數（考試直接考）

- **MSE (Mean Squared Error)**：L = (1/n) Sigma(y - y_hat)^2。
  - 用於迴歸任務。
  - 直覺：預測偏差的平方平均。

- **Cross-Entropy**：L = -Sigma y_i x log(y_hat_i)。
  - 用於分類任務。
  - 直覺：預測機率越偏離真實標籤，懲罰越重。

- **考試重點**：什麼任務用什麼損失函數。迴歸 = MSE，分類 = CE。

#### 導數（高中複習）

- **定義**：函數在某點的斜率 = 瞬間變化率。
- **例**：f(x) = x^2 → f'(x) = 2x。在 x=3 的斜率 = 6。
- **教學提醒**：只複習 1 張，確認學生記得「導數 = 斜率」就好。

#### 偏微分

- **定義**：多變數函數對其中一個變數的變化率，其他變數固定。
- **視覺**：3D 曲面上沿一個方向切出的斜率。
- **AI 意義**：loss 對單一權重 w_i 的變化率 = 告訴你「調這個權重，loss 會怎麼變」。

#### 梯度下降（核心）

- **梯度**：所有偏微分組成的向量，指向最陡上坡方向。
- **更新規則**：w = w - lr x gradient（沿負梯度走 = 下坡）。
- **直覺**：蒙眼站在山上，用腳探哪邊最陡，往下走一步。重複直到谷底。
- **教學提醒**：用球在碗裡滾動的動畫比喻。

#### Learning Rate（考試直接考）

- **lr 太大**：步伐太大 → 左右震盪 → 跳過最低點 → 不收斂。
- **lr 太小**：步伐太小 → 收斂極慢 → 可能卡在局部最小。
- **lr 剛好**：穩定下降 → 到達最低點。
- **視覺**：三面板比較圖（考試常考看圖判斷 lr 問題）。

#### 鏈式法則 = 反向傳播

- **公式**：dL/dw = dL/dz x dz/dw（一層一層傳回去）。
- **AI 意義**：反向傳播就是用鏈式法則算出 loss 對每個權重的梯度。
- **教學提醒**：只需 1 張概念圖，不需推導完整計算圖。考試不考反向傳播推導。

#### 凸性

- **凸函數**：碗形，只有一個最低點。線性回歸的 MSE = 凸函數 → 一定找到全域最佳。
- **非凸函數**：崎嶇地形，多個局部最小。神經網路的 loss = 非凸 → 可能卡住。
- **視覺**：碗 vs 崎嶇地形的對比圖。

#### 正則化（考試直接考）

- **L1 正則化 (Lasso)**：penalty = lambda x Sigma|w_i|。
  - 效果：鼓勵稀疏（很多權重變成 0）= 自動特徵選擇。
  - 幾何：約束區域是菱形（鑽石形），角落在軸上 → 解容易落在軸上 = 稀疏。

- **L2 正則化 (Ridge)**：penalty = lambda x Sigma w_i^2。
  - 效果：鼓勵小權重（但不會是 0）= 防止某個特徵主導。
  - 幾何：約束區域是圓形 → 解不太會在軸上 = 不稀疏但均勻縮小。

- **考試記法**：L1 = Lasso = 稀疏 = 鑽石。L2 = Ridge = 小權重 = 圓形。

## 4. 常見數學迷思（教學時需主動破除）

1. **「我數學很差所以學不會 AI」**：考試的數學深度大約是高中加強版，不需要大學微積分。
2. **「公式要背下來」**：理解含義 + 知道什麼時候用比背公式重要。考試考的是解讀，不是推導。
3. **「矩陣乘法很複雜」**：考試最多考 2x3 x 3x2，學會一個規則就夠了。
4. **「梯度下降很抽象」**：就是「蒙眼在山上找谷底」，用腳探路然後走一步。
5. **「偏微分很難」**：只需要知道「loss 對某個權重的敏感度」，不需要真的算。

## 7. 常見 Q&A

- **Q：只考初級需要讀這章嗎？**
  A：不需要。初級考試不考數學計算，只考概念判斷。這章是中級預備。

- **Q：考試會考偏微分的計算嗎？**
  A：不會。考試考的是概念——「梯度下降在做什麼」「learning rate 太大會怎樣」，不會要你算偏微分。

- **Q：矩陣乘法一定要會手算嗎？**
  A：是的。樣題直接考 2x3 x 3x2 的手算。但不會考更大的矩陣，熟練一個範例就夠了。

- **Q：L1 和 L2 要背公式嗎？**
  A：要知道公式的形式（絕對值 vs 平方），更重要的是記住效果差異——L1 稀疏、L2 小權重。

- **Q：貝氏定理的計算會考嗎？**
  A：有可能。建議熟練醫療篩檢的完整推導流程，數字會設計成可以手算的。
