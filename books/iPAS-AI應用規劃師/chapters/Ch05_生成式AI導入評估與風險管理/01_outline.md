# Ch05 講師講稿 — 生成式AI導入評估與風險管理

> 節次時長：120 分鐘（講授 75 + 考題演練 30 + 總結回顧 15）
> Governing thought：「導入 AI 不是裝一個工具，是做一個決策——考試考的是你的規劃判斷力。」
> 考試對應：初級 L1（導入評估）/ L2（導入規劃）/ L3（風險管理）

---

## 1. 本節目標

1. 能運用「技術可行性 x 商業價值 x 組織準備度」三維框架評估 AI 導入方案。
2. 熟悉模型效能評估指標（Accuracy / F1 / Latency / Cost），能在考題中正確配對。
3. 掌握成本效益分析（ROI / TCO）與隱性成本辨識，不被表面數字誤導。
4. 背熟經濟部《AI導入指引》五階段流程，這是送分題。
5. 區分 AI Agent 與傳統 AI 工具的差異，應對代理相關新題型。
6. 辨識 GenAI 三大風險維度（倫理/資安隱私/合規性），能判斷風險影響層級。

## 4. 關鍵教學點

### Part A：導入評估（L1）

#### A-1 導入評估三維度
- **技術可行性**：現有技術能否達到目標效能？資料量是否足夠？基礎設施是否到位？
- **商業價值**：導入後能降多少成本？提升多少效率？創造多少新營收？
- **組織準備度**：有沒有 AI 人才？管理層支不支持？員工接受度如何？
- **考試重點**：三維度缺一不可，考題常用「以下何者不屬於導入評估範疇」出題。

#### A-2 模型效能評估
- **Accuracy / Precision / Recall / F1**：分類任務基本指標。
- **Latency / Throughput**：即時性需求判斷。
- **Cost per query**：API 計費 vs 自建成本。
- **考試陷阱**：「準確率高 = 模型好」是錯的，要看場景（如醫療偏重 Recall）。

#### A-3 成本效益分析
- **ROI（投資報酬率）**：(收益 - 成本) / 成本。
- **TCO（總擁有成本）**：硬體 + 軟體 + 人力 + 維運 + 資料標註 + 退場成本。
- **隱性成本清單**：資料清洗、模型再訓練、員工教育訓練、法遵成本。
- **考試重點**：TCO 包含哪些項目是高頻考題。

### Part B：導入規劃（L2）

#### B-1 經濟部《AI導入指引》
- **五階段**：需求確認 → 可行性評估 → 概念驗證（PoC）→ 系統開發 → 部署維運。
- **每階段關鍵產出**：需求規格書 / 評估報告 / PoC 結果 / 系統文件 / SLA。
- **考試重點**：階段順序與各階段的核心任務，幾乎每次必考。

#### B-2 導入規劃五面向
- **目標設置**：SMART 原則，量化的 KPI。
- **資源分配**：人力、預算、時間、資料、基礎設施。
- **導入策略**：漸進式 vs 全面式、試點 vs 全面推廣。
- **因應措施**：風險預案、變更管理、stakeholder 溝通。
- **測試**：功能測試、壓力測試、使用者驗收測試（UAT）。

#### B-3 AI Agent 導入
- **什麼是 AI Agent**：具備感知-推理-行動循環的自主系統。
- **與傳統 AI 工具差異**：被動回應 vs 主動規劃、單次執行 vs 多步驟決策。
- **導入考量**：自主權邊界、人機協作模式、監控與介入機制。
- **考試趨勢**：AI Agent 是 2025-2026 新增考題方向，需特別留意。

### Part C：風險管理（L3）

#### C-1 GenAI 三大風險維度
- **倫理風險**：偏見與歧視、透明度不足、問責模糊、深偽（Deepfake）。
- **資料安全與隱私**：資料外洩、模型竊取、membership inference、PII 暴露。
- **合規性**：個資法、著作權法、AI 基本法草案、歐盟 AI Act 對照。

#### C-2 風險影響矩陣
- 橫軸：發生機率（低/中/高）。
- 縱軸：衝擊程度（低/中/高）。
- 九宮格分類：接受/監控/緩解/迴避。
- **考試重點**：能判斷特定風險屬於哪個象限，並選出正確的應對策略。

#### C-3 風險緩解策略
- **預防**：設計階段納入倫理審查、資料脫敏、存取控制。
- **偵測**：持續監控、異常偵測、定期稽核。
- **回應**：事件應變計畫、通報機制、緊急停機。
- **復原**：回復計畫、影響評估、改善措施。

## 5. 學員常犯錯誤（考試易錯點）

1. **導入評估 = 技術評估**：忘記組織面與商業面，選擇題直接被誘答。
2. **ROI 與 TCO 混淆**：ROI 是比率，TCO 是總成本金額，方向不同。
3. **《AI導入指引》階段順序記錯**：需求確認必在最前面，不是可行性評估。
4. **風險管理 = 資安**：考題會出倫理與合規的情境，只會資安拿不到分。
5. **AI Agent 題型不熟**：新題型，容易因為沒準備直接放棄。
