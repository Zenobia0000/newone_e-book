# 課程總計畫：iPAS AI 應用規劃師備考實戰

**版本：** 1.0  
**總時數：** 30 小時（10 章）  
**對應證照：** 經濟部 iPAS AI 應用規劃師（初級 + 中級）  
**時間定錨：** 115 年度考試

---

## 一、課程定位

本課程是 **iPAS AI 應用規劃師能力鑑定** 的完整備考課程，精準對齊官方評鑑內容範圍（115.02 版），涵蓋初級 2 科 + 中級 3 科的所有考試主題。

**設計原則：**
1. **評鑑範圍全覆蓋**：每章的 `03_exam_analysis.md` 逐條對照 L-code
2. **考題導向**：不只教概念，更教「考試怎麼考、陷阱在哪」
3. **模擬實戰**：每章含 5-10 道模擬題，中級科目2/3 含 Python 程式題
4. **分級路線**：Ch01-05 覆蓋初級，Ch06-10 覆蓋中級，可依需求選讀

---

## 二、考試規則速查

### 初級（2 科）
| 項目 | 內容 |
|------|------|
| 科目 1 | 人工智慧基礎概論（L11） |
| 科目 2 | 生成式 AI 應用與規劃（L12） |
| 題型 | 每科 50 題單選，75 分鐘 |
| 及格 | 同時報考：平均 ≥ 70 且每科 ≥ 60；分開報考：每科 ≥ 70 |
| 成績保留 | 單科 ≥ 70 保留 3 年 |
| 證書效期 | 永久有效 |
| 115 年日期 | 03/21、05/16、08/15、11/07 |

### 中級（科目 1 必考 + 科目 2 或 3 擇一）
| 項目 | 內容 |
|------|------|
| 科目 1 | 人工智慧技術應用與規劃（L21）— 必考 |
| 科目 2 | 大數據處理分析與應用（L22）— 選考 → 取得「數據分析」證書 |
| 科目 3 | 機器學習技術與應用（L23）— 選考 → 取得「機器學習」證書 |
| 題型 | 每科 50 題單選，90 分鐘 |
| 程式題 | 科目 2/3 約 25% Python 程式相關題型 |
| 及格 | 每科 ≥ 70 |
| 成績保留 | 單科 ≥ 70 保留 3 年 |
| 證書效期 | 5 年（需換證） |
| 115 年日期 | 05/23、11/14 |
