## 考試規則速查

### 初級（2 科，每科 50 題單選，75 分鐘）
- 科目 1：人工智慧基礎概論
- 科目 2：生成式 AI 應用與規劃
- 及格：同時報考 → 平均 70 分且每科 ≥ 60 分；分開報考 → 每科 ≥ 70 分
- 單科成績保留 3 年
- 證書永久有效
- 115 年考試：03/21、05/16、08/15、11/07

### 中級（科目 1 必考 + 科目 2 或 3 擇一，每科 50 題單選，90 分鐘）
- 科目 1：人工智慧技術應用與規劃（必考）
- 科目 2：大數據處理分析與應用（選考）→ 取得「AI 應用規劃師（數據分析）」
- 科目 3：機器學習技術與應用（選考）→ 取得「AI 應用規劃師（機器學習）」
- 科目 2/3 含 Python 程式題約 25%
- 每科 ≥ 70 分及格
- 單科成績保留 3 年
- 證書效期 5 年（需換證）
- 115 年考試：05/23、11/14
