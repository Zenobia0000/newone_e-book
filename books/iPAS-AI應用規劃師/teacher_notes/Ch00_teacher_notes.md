## 3. 關鍵教學點 (Key Teaching Points)

### Part A：機率與統計基礎（60 min）

1. **離散 vs 連續的判斷標準要講清楚**：離散可列舉（骰子 1-6），連續不可列舉（身高 170.32...）。離散用 P(X=x)，連續用 f(x)dx。不需教積分計算，只要學員知道「連續不能問 P(X=某值)」。

2. **條件機率方向是最常見考試陷阱**：P(A|B) 不等於 P(B|A)。P(有病|陽性) 不等於 P(陽性|有病)。用「你最想知道的」vs「醫學文獻寫的」來區分。

3. **貝氏定理用醫療篩檢跑完整推導**：盛行率 1%、靈敏度 99%、特異度 95%，P(有病|陽性) 只有約 17%。這個案例要讓學員親手算一次。教學重點：盛行率低時陽性結果的真實性也不高。

4. **期望值連結損失函數，變異數連結過擬合**：E(X) = 加權平均 = 損失函數的基礎。Var 大 = 不穩定 = overfit。用幾何直覺教：期望值 = 分佈中心，變異數 = 分佈寬度。

5. **Type I/II 錯誤用比喻固定記憶**：Type I = 喊「狼來了」（假陽性），Type II = 放走壞人（假陰性）。p-value < alpha → 拒絕 H0。考試直接考情境判斷。

### Part B：線性代數（50 min）

6. **向量 = 一筆資料的所有特徵排成一列**：用試算表比喻——一行資料 = 一個向量。2D 中 [3,4] = 從原點到 (3,4) 的箭頭。

7. **內積手算必須現場練一次**：[1,2,3]·[4,5,6] = 4+10+18 = 32。連結線性回歸 y_hat = w·x + b。

8. **矩陣乘法是考試直接考的手算題**：(m,n)×(n,p)=(m,p)，n 必須一樣。用 2x3 × 3x2 的完整手算範例，確保學員會對齊維度。

9. **特徵值/特徵向量只考概念**：Av = λv，PCA = 找協方差矩陣最大特徵值方向。考試不考計算，只考「PCA 為什麼能降維」。

### Part C：最佳化與梯度下降（50 min）

10. **MSE vs Cross-Entropy 是送分題**：迴歸 = MSE，分類 = CE。考試直接考「什麼任務用什麼損失函數」。

11. **梯度下降用「蒙眼下山」比喻**：w = w - lr × gradient。用球在碗裡滾動的動畫描述。

12. **Learning rate 三面板是高頻考點**：太大→震盪、太小→收斂慢、剛好→穩定下降。考試常考看圖判斷 lr 問題。

13. **L1 vs L2 正則化用幾何記憶**：L1 = Lasso = 鑽石形約束 = 稀疏。L2 = Ridge = 圓形約束 = 小權重。

## 4. 學員常犯錯誤 (Common Pitfalls)

- **P(A|B) 和 P(B|A) 搞反**：條件機率的方向是考試最愛的陷阱。
- **矩陣乘法維度不會配**：(m,n) × (n,p) = (m,p)，中間的 n 必須一樣，結果取兩端。
- **Learning rate 太大的後果**：會震盪跳過最低點，不是「學得更快」。
- **Type I 和 Type II 錯誤搞混**：Type I = 假陽性（喊狼來了），Type II = 假陰性（放走壞人）。
- **L1 和 L2 效果搞反**：L1 製造稀疏（鑽石形），L2 縮小權重（圓形）。
- **「我數學很差所以學不會 AI」**：考試深度大約是高中加強版，不需要大學微積分。

## 5. 提問設計 (Discussion Prompts)

1. 你的 COVID 快篩陽性了，你覺得你真的有 COVID 的機率是多少？大多數人的直覺會說 90% 以上。我們來算算看。（→ 引出貝氏定理）
2. 假設你是一個 AI 模型，你的任務是猜明天的氣溫。你的損失函數是什麼？你會怎麼定義「猜錯的代價」？（→ MSE）
3. 如果你站在一座 3D 山上被蒙住眼睛，你怎麼走到谷底？你會怎麼決定每一步的方向和大小？（→ 梯度下降）

## 7. 常見 Q&A

**Q1：只考初級需要讀這章嗎？**
A：不需要。初級考試不考數學計算，只考概念判斷。這章是中級預備。

**Q2：考試會考偏微分的計算嗎？**
A：不會。考的是概念——「梯度下降在做什麼」「learning rate 太大會怎樣」，不會要你算偏微分。

**Q3：矩陣乘法一定要會手算嗎？**
A：是的。樣題直接考 2x3 × 3x2 的手算。但不會考更大的矩陣，熟練一個範例就夠了。

**Q4：L1 和 L2 要背公式嗎？**
A：要知道公式的形式（絕對值 vs 平方），更重要的是記住效果差異——L1 稀疏、L2 小權重。

**Q5：貝氏定理的計算會考嗎？**
A：有可能。建議熟練醫療篩檢的完整推導流程，數字會設計成可以手算的。
